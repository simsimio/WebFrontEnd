Retina Ready Logo and other images

Tides includes some javascript to provide full retina ready support, here's what you need to do;

  1. Add an image and the retina version to your /img/ folder, they need to be named like this; example.jpg & example@2x.jpg
  2. Add a class of retina to your image tag, the javascript will now search for a retina version of this image and will either return with the retina version for retina displays, or will return a 404 if the image cannot be found.
Type Code
Image <img src="img/logo.jpg" alt="logo" class="retina" />

Menu Active Classes

The main nav and side-menu nav active states are handled by javascript. If you want to do this manually, you'd add a class of active to the containing li element of the active a tag.

To use the scroll spy technique, which scrolls you through a page, your nav items need a class of scroller

If you leave off the scroller class then your nav items will simply links like normal link tags.

Menu

Tides uses a very traditional system for the menu, shown to the right.

All styles for the menu will be found in /css/style.css

Changing Fonts

If you would like to use different Google Fonts, copy the font link in Google Fonts page and paste it before closing head tag in your HTML files.


Then you need to add the font name to your CSS styles.

body {
    font-family: 'Lato', sans-serif;
}
